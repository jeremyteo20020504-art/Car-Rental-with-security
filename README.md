MotoZoom is the website name of the Car Rental platform. 
The program is hosted on Apache and MySQL database, requiring a download of Apache, if it has not already been done so.

MotoZoom folder to be placed within the .htdocs to allow Apache to read the code and load the website information.

The following are the database to be uploaded on the hosted MySQL database used:
bookingall.sql
carsallofthem.sql

While the website will run without the relevant information, it is essential to get the intended experience.

sendmail folder to replace the entire section for simplicity, as the option is pre-configured.

php.ini to be replaced the existing one in php > php.ini
Current configuring contains email credentials to allow website to send emails.



"bookingall" and "carsallofthem" are database folders, to be stored within mysql > data
No file overwriting required, to load database info.
